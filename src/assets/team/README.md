# Team Photo Folders

Drop each member's photo in the matching portfolio folder.
**File naming:** use the member's index as the filename, e.g. `1.jpg`, `2.jpg` etc.  
Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

```
src/assets/team/
├── marketing/              ← 5 members  → 1.jpg … 5.jpg
├── documentation/          ← 5 members  → 1.jpg … 5.jpg
├── poster-designing/       ← 4 members  → 1.jpg … 4.jpg
├── hospitality/            ← 5 members  → 1.jpg … 5.jpg
├── treasury/               ← 4 members  → 1.jpg … 4.jpg
├── web-designing/          ← 3 members  → 1.jpg … 3.jpg
└── social-media-promotions/← 4 members  → 1.jpg … 4.jpg
```

> When a photo is missing the card will show a grey placeholder—no errors will occur.
